This REST server simulates the classic Memory Game.

Start a game by sending a POST request with name and email. You will get an ID back which you use to make additional request. You will also get the size of the board.

Now you could draw the board and place all the cards (width×height) in a face-down manor.

Get a value of a card by sending a request providing the game ID and x,y coordinates of a card.

Allow only two cards to be shown at a time

When there is a match, make the pair immutable

You can send a request at any time to end the game, you only have to provide the coordinates of what you think are the last two cards, as well as the ID of the game.

Once you have ended the game you can not make any more moves
